# 🇫🇷 French Learning Tracker

A personal French course tracker — upload your lesson PowerPoints to Claude, get structured JSON, and view everything in a clean GitHub Pages dashboard.

`https://YOUR_USERNAME.github.io/french-learning-tracker`

## Structure

```
docs/           → GitHub Pages site (index.html)
lessons/        → One JSON file per lesson
DAILY_WORKFLOW.md → Your 15-min daily routine
```
